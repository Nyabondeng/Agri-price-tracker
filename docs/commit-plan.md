# Suggested Commit Sequence

Use these example commit messages exactly or adapt them.

1. `docs: define Ghana context, users, and project scope in README`
2. `chore: add secure gitignore and MIT license`
3. `feat: add initial Node.js server with crop price API`
4. `feat: add simple web page for displaying crop prices`
5. `docs: add GitHub board backlog and branch protection checklist`

No emojis should be used in commit messages.
